🚀 QUICK COMMANDS

   $ pip install -r requirements.txt

2. SET GITHUB TOKEN
   $ export GITHUB_TOKEN=ghp_your_token_here

3. RUN BASIC EVALUATION
   $ python main.py --org your-org --pattern "assignment-"

4. EVALUATE WITH CUSTOM DATE RANGE
   $ python main.py --org your-org --start-date 2024-09-01 --end-date 2024-09-30

5. EXPORT ONLY CSV
   $ python main.py --org your-org --formats csv

6. USE CUSTOM CONFIG
   $ python main.py --config config/custom.yaml --org your-org

7. GET HELP
   $ python main.py --help

================================================================================
📊 EVALUATION SYSTEM
================================================================================

5 Independent Metrics (0-100 each):

1. COMMIT FREQUENCY (25% weight)
   • How often students commit
   • Ideal: 5-10 commits per contributor
   • Flags: Solo work or very few commits

2. COMMIT MESSAGE QUALITY (25% weight)
   • Message length and clarity
   • Minimum: 10 characters
   • Good: 50+ characters
   • Penalizes: Vague messages ("fix", "update")

3. INDIVIDUAL CONTRIBUTION (30% weight)
   • Fair distribution of work
   • Uses Gini-coefficient-like analysis
   • Flags: >80% by one person
   • Penalizes: Solo work

4. BRANCH STRATEGY (10% weight)
   • Proper use of branches
   • Rewards: Multiple branches, feature branches
   • Evaluates: Git workflow compliance

5. DOCUMENTATION (10% weight)
   • README.md presence
   • Repository description
   • Completion scoring

FINAL GRADE:
├─ A: 90-100 (Excellent)
├─ B: 80-89 (Good)
├─ C: 70-79 (Satisfactory)
├─ D: 60-69 (Passing)
└─ F: <60 (Failing)

================================================================================
🆘 TROUBLESHOOTING
================================================================================

Problem: "GitHub token not provided"
Solution: export GITHUB_TOKEN=ghp_your_token_here

Problem: "Organization not found"
Solution: Verify org name (case-sensitive)
         curl -H "Authorization: token $GITHUB_TOKEN" https://api.github.com/orgs/your-org

Problem: "No repositories found"
Solution: Check repository naming pattern matches your repos
         python main.py --org your-org --pattern ""

Problem: API rate limit exceeded
Solution: GitHub allows 5000 requests/hour
         ~10 requests per student
         Safe for 500+ students per hour

For more help: See QUICKSTART.md or README.md
